# Solar-Powered Environmental & Soil Monitor

An ultra-low-power environmental telemetry node built around a custom ESP32-C3 PCB. The system implements a split-power architecture to harvest solar energy, sample ambient and soil health parameters, and enter deep sleep to enable indefinite outdoor operation.

## Hardware Architecture & Pin Mapping

The custom PCB routes the internal hardware peripherals of the ESP32-C3-WROOM-02U module to external components using these exact net configurations:

| Peripheral Interface | ESP32-C3 Pin Name | PCB Designator | Functional Notes |
| :--- | :--- | :--- | :--- |
| **USB Native Data (-)** | GPIO18 (IO18) | J4 (A7/B7) | Hardwired to Type-C D- line for native flashing |
| **USB Native Data (+)** | GPIO19 (IO19) | J4 (A6/B6) | Hardwired to Type-C D+ line for native flashing |
| **I2C Serial Data (SDA)** | GPIO2 (IO2) | J2 (Pin 3) | Routes directly to STEMMA QT / Qwiic SHT41 Port |
| **I2C Serial Clock (SCL)** | GPIO10 (IO10) | J2 (Pin 4) | Dedicated clock channel overridden in code |
| **Analog Soil Signal** | GPIO1 (IO1) | J3 (Pin 3) | Wired to ADC1_CH1 for 12-bit moisture sampling |
| **Boot Mode Strap** | GPIO9 (IO9) | R1 / R3 | Tied via 10k Ohm to 3V3 to force execution from Flash |
